# 计算属性、方法与侦听器

产生的原因：

<div id="app">
  {{fullName}}
</div>
<script type="text/javascript">
  var vm = new Vue({
    el: "#app",
    data: {
      firstName: 'feng',
      lastName: 'zi',
      fullName: "fengzi"
    }
  })
</script>

这个代码能在页面上显示出fullName，但是会使得代码冗余

# 计算属性

<div id="app">
  {{fullName}}
</div>
<script type="text/javascript">
  var vm = new Vue({
    el: "#app",
    data: {
      firstName: 'feng',
      lastName: 'zi'
    },
    computed: {
      fullName: function () {
        return this.firstName + " " + this.lastName
      }
    }
  })
</script>

计算属性是内置缓存的，如果计算属性依赖的数据没有发生变化，就不会重新执行一次。

<div id="app">
  {{fullName}} {{age}}
</div>
<script type="text/javascript">
  var vm = new Vue({
    el: "#app",
    data: {
      firstName: 'feng',
      lastName: 'zi',
      age: 28
    },
    computed: {
      fullName: function () {
        console.log('计算了一次')  ;
        return this.firstName + " " + this.lastName
      }
    }
  })
</script>

# 方法---并没有缓存机制

<div id="app">
  {{fullName()}} {{age}} // 注意这里的括号
</div>
<script type="text/javascript">
  var vm = new Vue({
    el: "#app",
    data: {
      firstName: 'feng',
      lastName: 'zi',
      age: 28
    },
    methods: {
      fullName: function () {
        return this.firstName + " " + this.lastName
      }
    }
  })
</script>

# 侦听器——同样也有缓存机制s

<body>
  <div id="app">
    {{fullName}} {{age}}
  </div>
  <script type="text/javascript">
    var vm = new Vue({
      el: "#app",
      data: {
        firstName: 'feng',
        lastName: 'zi',
        fullName: 'feng zi',
        age: 28
      },
      watch: {
        firstName: function () {
          this.fullName = this.firstName + " " + this.lastName
        },
        lastName: function () {
          this.fullName = this.firstName + " " + this.lastName
        }
      }
    })
  </script>
</body>

# 计算属性的getter和setter

<div id="app">
  {{fullName}}
</div>
<script type="text/javascript">
  var vm = new Vue({
    el: "#app",
    data: {
      firstName: 'feng',
      lastName: 'zi'
    },
    computed: {
      fullName: {
        get: function () {
          return this.firstName + " " + this.lastName
        },
        set: function (value) {
          var arr = value.split(" ")
          this.firstName = arr[0]
          this.lastName = arr[1]
        }
      }
    }
  })
</script>

# vue的条件渲染

<div id="app">
  <div v-if="show" data-test="v-if">{{message}}</div>
  <div v-else data-test="v-else">Bye Feng</div>
  <div v-show="show" data-test="v-show">{{message}}</div>
</div>
<script type="text/javascript">
  var vm = new Vue({
    el: "#app",
    data: {
      show: false,
      message: 'Hello Fengzi'
    }
  })
</script>

<div id="app">
  <div v-if="show === 'a'" >This is A</div>
  <div v-else-if="show === 'b'">This is B</div>
  <div v-else>This is others</div>
</div>
<script type="text/javascript">
  var vm = new Vue({
    el: "#app",
    data: {
      show: "a"
    }
  })
</script>

下面的代码中，如果不加入key值，当show变化的时候，两个DOM都会变化。因为Vue中会尝试服用以前的标签。。。加入key值就不会这样，Vue会知道这是唯一的标签

<div id="app">
  <div v-if="show">
    用户名：<input type="text" key="username">
  </div>
  <div v-else>
    邮箱名：<input type="text" key="email">
  </div>
</div>
<script type="text/javascript">
  var vm = new Vue({
    el: "#app",
    data: {
      show: false
    }
  })
</script>

第二种方法就是改变数据的引用的地址，vm.list = [{id: "0101010101",text: "hello1"},{id: "0101010102",text: "feng1"}, {id: "0101010103",text: "zi"}]

# vue中的set方法

还是前面的代码，若要改变可以使用set方法：Vue.set(vm.userInfo, "address", "beijing") 这样在页面中就会显示出五条数据

还可以使用vue实例中的set方法进行改变：vm.$set(vm.userInfo, "address", "beijing")

<div id="app">
  <div v-for="(item, index) of userInfo">
    {{item}}
  </div>
</div>
<script type="text/javascript">
  var vm = new Vue({
    el: "#app",
    data: {
      userInfo: [1, 2, 3, 4]
    }
  })
</script>

上面的代码中，若要数组改变页面跟着改变，除了上面提到的两种方法，还有一种就是使用set方法： Vue.set(vm.userInfo, 1, 5) 或 vm.$set(vm.userInfo, 1, 10)

# 使用组件的细节

<div id="app">
  <table>
    <tbody>
      <row></row>
      <row></row>
      <row></row>
    </tbody>
  </table>
</div>
<script type="text/javascript">
  Vue.component('row', {
    template: '<tr><td>this is a row</td></tr>'
  })
  var vm = new Vue({
    el: "#app"
  })
</script>

上面的代码中，在页面显示的是三行this is a row。但若要在控制台中查看元素，就会发现tr标签没有在tbody中。。解决这样的bug，可以使用Vue提供给的is属性-----这样就不回出bug了

<div id="app">
  <table>
    <tbody>
      <tr is="row"></tr>
      <tr is="row"></tr>
      <tr is="row"></tr>
    </tbody>
  </table>
</div>
<script type="text/javascript">
  Vue.component('row', {
    template: '<tr><td>this is a row</td></tr>'
  })
  var vm = new Vue({
    el: "#app"
  })
</script>

在子组件中使用data的时候，必须是一个函数，而不能是一个对象，让每一个自组件都拥有一个独立的数据存储。

<div id="app">
  <table>
    <tbody>
      <tr is="row"></tr>
      <tr is="row"></tr>
      <tr is="row"></tr>
    </tbody>
  </table>
</div>
<script type="text/javascript">
  Vue.component('row', {
    data: function () {
      return {
        content: 'this is content'
      }
    },
    template: '<tr><td>{{content}}</td></tr>'
  })
  var vm = new Vue({
    el: "#app"
  })
</script>

<div id="app">
  <counter ref="one" @change="handleChange"></counter>
  <counter ref="two" @change="handleChange"></counter>
  <div>{{total}}</div>
</div>
<script type="text/javascript">
  Vue.component('counter', {
    template: '<div @click="handleClick">{{number}}</div>',
    data: function () {
      return {
        number: 0
      }
    },
    methods: {
      handleClick: function () {
        this.number++
        this.$emit('change')
      }
    }
  })
  var vm = new Vue({
    el: "#app",
    data: {
      total: 0
    },
    methods: {
      handleChange: function () {
        this.total = this.$refs.one.number + this.$refs.two.number
      }
    }
  })
</script>

# 父子组件的数据传递

父组件通过属性的方式向自组件发送数据

单项数据流---子组件不能修改父组建传递过来的参数

<div id="app">
  <counter :count="1" @inc="handleIncrease"></counter>
  <counter :count="2" @inc="handleIncrease"></counter>
  <div>{{total}}</div>
</div>
<script type="text/javascript">
  var counter = {
    props: ['count'],
    data: function () {
      return {
        number: this.count // 克隆一个副本，更改副本
      }
    },
    template: '<div @click="handleClick">{{number}}</div>',
    methods: {
      handleClick: function() {
        this.number = this.number + 2
        this.$emit('inc', 2)
      }
    }
  }
  var vm = new Vue({
    el: "#app",
    data: {
      total: 3
    },
    components: {
      counter: counter
    },
    methods: {
      handleIncrease: function (step) {
        this.total += step
      }
    }
  })
</script>
